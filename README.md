# Echoes of Innocence

 Echoes of Innocence is a text-based detective adventure game where you explore rooms, collect evidence, and try to prove your client's innocence.

---

## Important Note

⚠️ **Use the code in the `Master` branch**. Do **not** use the `Main` branch. The Master branch contains the most up-to-date and fully working version of the game.

---

## How to Run the Game

1. **Download all the files** from the `Master` branch. Make sure you have all `.java` files (Game.java, Room.java, Item.java, Inventory.java, Player.java, and the room classes).

2. **Create a new Java project** on your local machine using your preferred IDE (like Eclipse or jGRASP).

3. **Add all of the downloaded files** to your project.

4. Once everything is added, **run the `Game.java` file** to start the game.

---

## Enjoy the Game!

Explore the rooms, inspect items, collect evidence, and try to prove your client's innocence. Good luck, detective!
